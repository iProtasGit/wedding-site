# Приглашение на свадьбу (Web-приложение)

Это современное веб-приложение для свадебного приглашения. Оно состоит из красивого сайта (где гости видят информацию и заполняют форму RSVP) и скрытой серверной части, которая автоматически сохраняет ответы гостей прямо в вашу таблицу Google Sheets, а также отправляет уведомления в **Telegram**!

Это руководство написано максимально подробно, шаг за шагом, чтобы вы смогли запустить проект, даже если никогда раньше не занимались программированием.

## Шаг 1. Что нужно установить перед началом

Чтобы запустить этот код на вашем компьютере, вам понадобятся две программы: одна для работы сайта (JavaScript), другая для работы сервера (Go).

1. **Скачайте и установите Node.js** (нужен для сборки красивого сайта):
   - Перейдите на сайт: [https://nodejs.org/](https://nodejs.org/)
   - Скачайте версию, на которой написано **"LTS"** (Recommended for most users).
   - Установите программу, нажимая "Далее" (Next) со стандартными настройками.

2. **Скачайте и установите Go** (язык программирования для серверной части):
   - Перейдите на сайт: [https://go.dev/dl/](https://go.dev/dl/)
   - Скачайте установщик для вашей операционной системы (Windows, macOS или Linux).
   - Установите программу со стандартными настройками.

## Шаг 2. Настройка Telegram Бота (Уведомления)

Мы хотим, чтобы при заполнении формы вам в Telegram сразу приходило уведомление.

1. Откройте приложение Telegram на телефоне или компьютере.
2. В поиске найдите бота **@BotFather** (с официальной синей галочкой).
3. Напишите ему команду `/newbot`.
4. Придумайте название бота (например, `Артём и Ульяна Свадьба`) и отправьте.
5. Придумайте юзернейм (обязательно должен заканчиваться на `bot`, например `ArtemUlyanaWedding_bot`).
6. В ответ BotFather пришлет длинный токен доступа, который выглядит примерно так: `1234567890:ABCDefGhIjKlMnOpQrStUvWxYz`. **Скопируйте его**. Это будет ваш `tgBotToken`.

**Как получить ваш Chat ID:**
Чтобы бот знал, кому отправлять сообщения, ему нужен ваш личный номер (Chat ID).
1. В поиске Telegram найдите бота **@getmyid_bot**.
2. Нажмите "Старт" (Start).
3. Он пришлет ваш `Your user ID` (например, `123456789`). **Скопируйте его**. Это будет ваш `tgChatId`.

*(О том, куда вставлять эти данные, написано в Шаге 5).*

## Шаг 3. Настройка Google Cloud (Создание бота для таблиц)

Чтобы сервер мог сам, без вашего участия, записывать ответы гостей в вашу Google Таблицу, ему нужен специальный электронный пропуск.

1. Перейдите в [Консоль Google Cloud](https://console.cloud.google.com/) и войдите под своим Google-аккаунтом.
2. **Создайте проект**:
   - В самом верху страницы (слева от строки поиска) нажмите на выпадающий список проектов (или кнопку "Select a project").
   - В появившемся окне в правом верхнем углу нажмите **"New Project"** (Новый проект).
   - Назовите его, например, `Wedding-App` и нажмите **"Create"** (Создать).
3. **Включите API для работы с таблицами**:
   - Убедитесь, что в верхнем меню выбран ваш новый проект.
   - В меню слева (три полоски) выберите **"APIs & Services"** -> **"Library"** (Библиотека).
   - В строке поиска введите `Google Sheets API`.
   - Нажмите на результат и затем на синюю кнопку **"Enable"** (Включить).
4. **Создайте сервисный аккаунт (нашего бота)**:
   - Снова откройте левое меню (три полоски), перейдите в **"APIs & Services"** -> **"Credentials"** (Учетные данные).
   - Наверху нажмите **"+ CREATE CREDENTIALS"** и выберите **"Service account"** (Сервисный аккаунт).
   - Придумайте имя (например, `rsvp-bot`) и нажмите **"Create and Continue"** (Создать и продолжить).
   - Следующие шаги (выбор роли) можно пропустить — просто прокрутите вниз и нажмите **"Done"** (Готово).
5. **Получите ключ (файл с паролем)**:
   - Вы вернетесь на страницу "Учетные данные". Внизу в разделе "Service Accounts" вы увидите созданного бота. Нажмите на его Email (он выглядит как длинный адрес: `rsvp-bot@...iam.gserviceaccount.com`).
   - Перейдите на вкладку **"KEYS"** (Ключи) (она находится наверху под названием аккаунта).
   - Нажмите **"ADD KEY"** -> **"Create new key"**.
   - Выберите формат **"JSON"** и нажмите **"Create"**.
   - На ваш компьютер скачается файл. **Это очень важно!** Найдите этот скачанный файл, переименуйте его ровно в `credentials.json` (маленькими буквами) и переместите его в папку `backend` внутри нашего проекта.

## Шаг 4. Настройка Google Таблицы

Теперь нам нужно создать саму таблицу и разрешить нашему "боту" в неё писать.

1. Откройте [Google Таблицы](https://sheets.google.com/) и создайте новую пустую таблицу.
2. В первой строке создайте три столбца (заголовки). Напишите в ячейке A1: `Имя гостя`, в B1: `Алкоголь`, в C1: `Нужен трансфер?`.
3. **Дайте доступ боту**:
   - Откройте скачанный ранее файл `credentials.json` (можно открыть в Блокноте).
   - Найдите там строчку, которая начинается с `"client_email":` и скопируйте адрес электронной почты внутри кавычек (например: `rsvp-bot@wedding-app-123.iam.gserviceaccount.com`).
   - Вернитесь в вашу Google Таблицу. В правом верхнем углу нажмите большую зеленую кнопку **"Настройки доступа"** (Share).
   - Вставьте скопированный email в поле добавления пользователей.
   - Обязательно убедитесь, что справа выбрано **"Редактор"** (Editor). Нажмите "Отправить". Теперь наш бот имеет право изменять эту таблицу.
4. **Скопируйте ID таблицы**:
   - Посмотрите на адресную строку вашего браузера, где открыта таблица. Она выглядит примерно так:
     `https://docs.google.com/spreadsheets/d/1Xy_АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ12345/edit#gid=0`
   - Скопируйте длинный набор букв и цифр между `/d/` и `/edit`. Это и есть **Spreadsheet ID** (в данном примере: `1Xy_АБВГДЕЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ12345`).

## Шаг 5. Настройка текстов и параметров сайта

В нашем проекте все тексты (ваши имена, дата, время, адреса) вынесены в отдельные файлы настроек.

### 1. Настройка сервера (`backend/config.json`)
Откройте файл `config.json`, который лежит в папке `backend`. Вставьте туда скопированный ID таблицы, а также данные Telegram-бота:
```json
{
  "port": ":8080",
  "spreadsheetId": "ВСТАВЬТЕ_СЮДА_СКОПИРОВАННЫЙ_ID_ТАБЛИЦЫ",
  "credentialsFile": "credentials.json",
  "tgBotToken": "ВСТАВЬТЕ_ТОКЕН_ОТ_BOTFATHER",
  "tgChatId": "ВСТАВЬТЕ_ВАШ_ID_ОТ_GETMYID_BOT"
}
```
*(Кавычки должны остаться, замените только текст внутри них)*

### 2. Настройка сайта (`frontend/public/config.json`)
Откройте файл `config.json`, который лежит по пути `frontend/public/config.json`. Здесь вы можете поменять все тексты приглашения:
- `names`: Ваши имена (например, "Артем & Ульяна").
- `weddingDate`: Точная дата и время в техническом формате (для таймера обратного отсчета). Пример: `2026-08-08T15:00:00`.
- `dateText`: Дата свадьбы текстом.
- `timeText`: Время начала.
- `venue`: Название ресторана и адрес.
- `transferInfo`: Информация про трансфер.
- `costumeInfo`: Информация про дресс-код.

Отредактируйте эти поля под ваше мероприятие.

## Шаг 6. Запуск проекта

У нас две части: frontend (лицевая часть сайта) и backend (сервер).

Для этого нам понадобится программа "Терминал" (Командная строка в Windows, Terminal в macOS).

### 1. Сборка сайта (Frontend)
Вам нужно превратить исходный код в готовые HTML-файлы.
1. Откройте терминал.
2. Перейдите в папку `frontend` нашего проекта. Если проект лежит на диске C в папке `d/fl`, команда будет такой:
   ```bash
   cd C:/d/fl/frontend
   ```
   *(На Mac: `cd /путь/к/папке/frontend`)*
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Соберите сайт:
   ```bash
   npm run build
   ```
   *После окончания появится папка `out`. В ней лежат готовые файлы.*

### 2. Запуск сервера (Backend)
1. В том же терминале перейдите в папку `backend`:
   ```bash
   cd ../backend
   ```
2. Скачайте библиотеки для Go:
   ```bash
   go mod tidy
   ```
3. Запустите сервер:
   ```bash
   go run cmd/server/main.go
   ```
   В терминале появится надпись:
   `Server starting on port :8080`
   и `Telegram bot integration enabled`.

### 3. Проверка!
1. Откройте браузер.
2. В адресной строке введите: `http://localhost:8080`
3. Нажмите Enter.

Вы должны увидеть ваше красивое свадебное приглашение!
Попробуйте заполнить форму RSVP и нажать "Отправить". Если все сделано верно:
1. Вы увидите красивое зеленое сообщение об успехе на сайте.
2. Ответ мгновенно появится в Google Таблице.
3. Вашему телефону придет уведомление в Telegram от вашего бота со всеми деталями гостя!
